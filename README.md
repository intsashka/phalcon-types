# phalcon-types
## Description
Types for phalcon 3.4.5

## Installation
In file composer.json add to section "repositories":
```
{
    "type": "git",
    "url": "git@github.com:intsashka/phalcon-types.git"
}
```

In file composer.json add to section "require":
```
"intsashka/phalcon-types": "*"
```

Run `composer update`
